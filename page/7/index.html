ubuntu下将apt源改为西电的源（仅校园网可以）

先备份apt源。

$ cd /etc/apt
$ sudo cp sources.list sources.list.old

用编辑器打开
$ sudo vi sources.list
然后用下面源的代替原来的内容，其中“precise”代表版本号，不同版本修改成相应的代号

deb ftp://linux.xidian.edu.cn/ubuntu/ precise main restricted universe multiverse
deb ftp://linux.xidian.edu.cn/ubuntu/ precise-backports main restricted universe multiverse
deb ftp://linux.xidian.edu.cn/ubuntu/ precise-proposed main restricted universe multiverse
deb ftp://linux.xidian.edu.cn/ubuntu/ precise-security main restricted universe multiverse
deb ftp://linux.xidian.edu.cn/ubuntu/ precise-updates main restricted universe multiverse

然后保存并退出:wq
测试一下
sudo apt-get update
可以看见刷刷刷出来了

<Image.Clip > 
    <RectangleGeometry Rect="x1 y1 width height"/>`
</Image.Clip>

Image.Clip中:

<RectangleGeometry Rect=”x1 y1 width height”/>

参数x1 y1代表以当前控件左上顶点为坐标原点，显示区域左上顶点的坐标，width height则是显示区域的宽和高。所以用:
