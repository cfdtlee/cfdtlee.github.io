Analytical Writing

May 12, 2014 | English

本文记录了在西安新东方学习GRE写作的笔记，GRE写作由西安新东方王宇飞老师任教。在GRE写作中logic and reasoning is important, 写作重推导，弱观点。

Requirment:

  • 文科专业：前30master，5分保险，4.5合格
  • 工科专业：前30master，4分保险，3.5合格
  • 艺术专业：

ISSUE

Analyze an issue task 149，六种问法（P45）题目紧凑，偏哲学性话题
450~650为宜

Argue题目长，但易读懂
450~650words

因果逻辑分析法

适用范围

特别适用issue题目中有因果逻辑关系的题目
P4T1, As时间＋因果关系
P4T1, Because
P4T5, Claim/Reason
P4T8

操作步骤

  1. 承认部分合理
  2. 分析题目中的原因是否成立（如果原因不成立，分析逻辑推导有问题）
  3. 即便题目中的原因成立，是否会导致题目中结果的相反方向（多个子方向）
  4. 即便题目中的原因成立，是否会导致题目中结果的无关方向（多个子方向）
  5. 分析题目中的原因是否一定能导致结果的出现（分析是否为充分条件）

举例
王老师的例子:因为学生上了王宇飞老师的作文课，所以学生们的作文分数一定能提升

  1. 王老师的课确实能提升分数
  2. 王老师是否认真备课？学生是否都来？
  3. 听了他的课分数会不会下降？
  4. 听了他的课可能分数没影响
  5. 是否只要老师讲课分数就一定能提高？还有诸多因素：学生有没有课后练习，考试那天状态如何等等。

举例P4T1
因为所有人们都已经越来越以来科技了，所以人们的思维能力一定会降低

  1. 承认合理性
    • 计算器－计算能力降低（中美中小学生计算能力，caculation table）
    • 输入法－书写记忆能力（汉字，单词拼写government restaurant environment）
    • 导航－识别能力
  2. 所有人？越来越？不能以全概偏（找非洲，肯尼亚，埃塞俄比亚（没水没电））
  3. 思维能力反而提高：
    • 开发者
    • 提高速度，扩展广度（宇宙探索，建筑，机械AutoCAD（画法几何，修改麻烦－>手工节省了花时间去做更复杂的形态）3D模型）
    • 使用科技出现问题要通过思考解决问题（如核能源出问题（切尔诺贝利、日本）），坏了怎么解决
  4. 会不会思维能力的不升高也不降低？当科技作为工具出现：交通、电灯、电脑、投影、话筒
  5. 是否需要结合其他条件才一定导致思维能力降低？完全依赖导致脑残、能力缺失，但如果注重思维训练就不会脑残

举例P4T4
丑闻是有用的，因为丑闻比任何改革者和发言人更能够引起我们对问题的关注

  1. 有用，水门事件，卡恩，克林顿，斯诺登
  2. 有没有可能改革者发言人更引起关注，宗教领袖（马丁路德金，南非国父曼德拉27年在监狱中度过，甘地），国家领袖
  3. 丑闻是有害的：对当事人是否有害（用老的观点看他），政治丑闻可能对社会发展有害（为了关一个人而去关，曼德拉被关）
  4. 丑闻是无用的，如娱乐圈的丑闻
  5. 分析充分性，文化氛围的辅助。意大利前总统orgy带着前杰克总理，老百姓关注了但没啥影响，

写作步骤：

  1. 写下各段的思维模版
  2. 填充例子
  3. 补充首尾段

为目的给建议

为了使得我们的国家繁荣昌盛，我们必须找出那些天才儿童，并加以最严格的培养。

操作步骤

  1. 承认部分合理性（大词细分）国家类话题分为政治（国执政）、经济（货币、福利、税收、股市1929美国金融危机）、文化（艺术、体育）三方面
  2. 分析建议是否可行（天才儿童不好找、难分辨，）
  3. 即便建议可行，是否会有副作用、危害等（对天才儿童身心伤害，对没选出来的也有危害）
  4. 分析建议达不到目标
  5. 分析建议是否唯一（必要条件分析，能源、外交、新加坡港口、意大利旅游业、武力）

举例P4T2
为了了解一个社会的主要特征，我们必须要研究这个社会的大城市

  1. 选择城市化比率高的国家新加坡、梵蒂冈（只有一个圣彼得教堂）、文化政治中心
  2. 并不是所有人去所有城市都很容易
  3. 研究的大城市不能代表整个国家（比如中国香港：政治游行、自由明主、经济发展模式、文化、教育、语言、开车）
  4. 很多文化保存在乡村，文化传统（过年百年初一到十五，但城市里并无完整地过春节）

ARGUE

  1. 题目特点：一篇建议80%，一篇科学结论

  2. 写作要求：

    • 首段总述错误
    • 中间段想起阐述错误，并且给出其他可能性（给作者的建议）
    • 尾端总结错误以及建议
      400~600

  3. 因果错误：

    • 识别因果错误：
      • 结论句推断因果错误（最重要的攻击点）
      • 提示词：because, since, so that, given that
      • 与此同时，在此之后while, after that
      • 无提示词
    • 如何攻击
      • 充分必要条件，该原因是否充分导致该结果的出现（抛开原文中的原因，根据目标和结果思考原因）
      • 反作用

  4. 数据错误：

    • 识别数据错误
      • 局部不代表全部（种类、数量、全部总数有多少？）没有给出调查数据（是否做了有效的、普遍性的调查）
      • 全部不代表局部
      • 百分比错误：关注数据统计的年份（背后的原因是什么）局部不代表全部，一年情况不能代表全部
      • 数据概念偷换
      • 数据过于模糊（非重点，more?投资产出比）
    • 对于任何（除了数据过于模糊）都要结合一下内容：分析数据是否准确、有效、具有真实性
      • 调查选项过少，是否选项(important)，是否有其他因素影响选择
      • 是否为主要调查(important)
      • 调查是否匿名
  5. 错误类比
    • 时间类比
    • 空间类比 P11-11
    • 情况类比 P10-4

攻击顺序

  1. 随机攻击
  2. 主次攻击：因果+类比，数据
  3. 让步式攻击：类比，数据+因果

笔记

  1. 写出三大类错误的名称
  2. 根据文章结论找出可能的建议，再对比原文中的建议（因果错误），分析不充分以及不必要
  3. 分析文章中其它的一些明显因果错误
  4. 通读全文，寻找其他错误并且归类
