JEE-say Hello World to Java EE

Goals

  1. Create Hello World web application using Java EE
  2. Deploy and test using Websphere Liberty Profile plug-in for Eclipse

Instructions

  1. Open Eclipse IDE

  2. Create new “Dynamic Web Project” using new project wizard

    a. Choose project name

    1. Select ‘New Runtime’ to be WebSphere Application Server Liberty Profile
    2. Choose installation directory (place where you saved liberty profile runtime)
    3. Uncheck Add projects to an EAR
    4. Finish
    5. Expand Web Module
    6. Under Servlet, add new Servlet
    7. Replace request mapping on top of the class to: @WebServlet( “/hello” )
    8. Replace code from doGet(…) method
      Response.getWriter().println( “Hello World!” );

  3. Deploy to Server & test
    a. Open Server view
    b. Select Websphere Liberty Profile
    c. Right click and add HelloWorld application
    d. Run
    e. Get the URL from server log
    f. Open browser within Eclipse
    g. Type in the URL
